NilBridge是一个在全平台下运行，提供QQ机器人协议库用于与Bedrock Dedicated Server通信的框架

## NilBridge的优点

-  基于Node.js，全平台兼容
- 全开源代码
- 性能优化极佳
- 文档完善
- 支持插件拓展
- 开箱即用
