假人管理器

# 命令

- /fp list   获取假人列表
- /fp add <假人名字> [是否允许聊天控制(是/否)]  添加一个假人
- /fp rem <假人名字>  移除一个假人
- /fp con <假人名字>  使假人连接到服务器
- /fp dis <假人名字>  使假人断开连接
- /fp disall   断开所有假人连接
- /fp conall   使所有假人连接到服务器
- /fp stall   获取所有假人的状态
